## Hoàn thành: Sửa lỗi Navigation Active State trên production

**Bug:** Trang web https://petrochemical-web.pages.dev/ luôn active tab "Trang chủ" cho mọi trang.
**Nguyên nhân:** Script cũ dùng `scrollY` để detect section, khi cuộn đến các section khác thì không match với hash nav links nên fallback về tab đầu tiên. Production cũng hardcode class "active" trong HTML.

**Cách sửa:** Inject script vào `live_check.html` (file đang chạy trên production):
- So sánh `window.location.pathname` với `href` của nav links
- Loại bỏ `.html` để so sánh linh hoạt (`/about` vs `about.html`)
- Dùng `classList.add/remove('active')` để set trạng thái
- Tested: tất cả 7 trang (index, about, products, services, gallery, career, contact) đều active đúng tab

**Source files đã update:**
- `homepage-xang-dau-hat-nhua.html` - full scroll-based logic
- `career.html`, `gallery.html` - simple path-based logic
- `live_check.html` - đang chạy production
